# github_api

Estudos e experimentos com a **GitHub API** para automação de tarefas de engenharia e produtividade.

## Objetivo
Explorar autenticação, consulta e atualização de recursos do GitHub para criar rotinas automatizadas (inventário, métricas, manutenção de repositórios).

## Conteúdo
- Notebooks de exploração da API
- Exemplos de consulta de metadados
- Base para automações futuras

## Stack
Python · Jupyter Notebook · REST API · GitHub CLI

## Evolução sugerida
- Transformar notebooks em scripts reutilizáveis
- Adicionar camada de configuração e logs
- Publicar utilitários de manutenção de portfólio
